# 🤝 Contributing to OverTheWire Bandit Walkthroughs
Thank you for your interest in contributing! This repository is a beginner-friendly, community-driven collection of walkthroughs and notes for the OverTheWire Bandit wargame. Whether you're fixing a typo or adding a full level walkthrough, every contribution counts.
## 👥 Who Can Contribute
Anyone! Whether you're:

- A complete beginner who just solved a level and wants to document it
- An experienced CTF player who wants to improve existing explanations
- A developer who wants to fix formatting, broken links, or typos
## 💡 Ways to Contribute
Here are the main ways you can contribute:

Types:
- 📝 Add a walkthrough -- Write a new level-XX.md for a missing level
- ✏️ Improve existing content -- Clearer explanations, better command breakdowns
- 🐛 Fix errors -- Wrong commands, broken links, typos
- 🔗 Add resources -- Helpful articles, man page links, YouTube tutorials
- 🎨 Improve formatting -- Consistent markdown, better structure

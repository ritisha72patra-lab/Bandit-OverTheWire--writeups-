# Bandit | OverTheWire-writeups
OverTheWire Bandit is the gateway wargame for anyone stepping into the world of cybersecurity and Linux. Aimed at absolute beginners, it teaches the foundational skills needed to tackle more advanced wargames organized across progressive levels where each solution unlocks the next. It's a CTF-style (Capture The Flag) wargame where each level hides a password you need to find in order to unlock the next one, all over SSH.

It covers real, practical skills:

- 🐧 Linux command-line navigation
- 🔐 File permissions and ownership
- 🔍 Searching and filtering with grep, find, sort
- 🌐 Networking basics with nc, openssl, nmap
- 📦 File compression and encoding (gzip, bzip2, base64, xxd)
- 🌿 Git internals and version control
- 🔑 SSH key authentication
## 🔨Tools and Technologies
- Linux Operating System or Ubuntu / Kali Linux VM
- Terminal
